表单
    作用:
        用于接收用户的数据并提交给服务器
    表单二要素
        1) form元素
            表单, 用于收集用户信息并提交给服务器
        2) 表单控件
            提供了能够与用户交互的可视化组件
    1. form元素
        用于接收用户的数据并提交给服务器
        form元素在网页中是不可见的, 但功能不能忽略
        标记:   <form></from>
        属性:
            action
                指定处理程序的地址
                不写默认是提交给本页
            method
                提交方式/方法, 提供了很多值, 其中有用的只有两个
                1) get 默认值
                    通常是在向服务器要数据时使用
                    特点:
                        (1) 提交的数据会显示在地址栏上
                        (2) 安全性较低
                        (3) 提交数据最大为2KB左右
                2) post
                    要将数据提交给服务器处理时使用
                    特点:
                        (1) 隐式提交, 看不到提交数据
                        (2) 安全性较高
                        (3) 无提交数据的大小限制
    2. 表单控件 ☭
        作用:
            提供了能够与用户交互的可视化组件
        注意:
            只有放在表单中的表单控件才以内需被提交
            表单控件都是"行内块"元素
        表单控件详解
            1) 文本框 & 密码框
                文本框 <input type="text">
                密码框 <input type="passwd">
            属性:
                name : 定义控件名称
                    提交给服务器使用, 如果没有name的话则无法提交
                value : 值
                    要提交给服务器的值, 同时也是默认能够显示在控件上的值
                maxlength
                    限制输入的最大字符数
                placeholder
                    占位符, 用户在未输入任何数据时所显示的内容
            2) 按钮
                提交按钮 <input type="submit">
                    将表单提交给服务器
                重置按钮 <input type="reset">
                    将表单内容恢复到初始化的状态
                普通按钮 <input type="button">
                    允许通过js自定义功能
                属性:
                    value : 按钮上显示的文本
                按钮标签
                    <button>内容</button>
                    属性: type
                    取值: submit/ reset/ button
            3) 单选按钮 & 复选框
                单选按钮 <input type="radio">
                复选框 <input type="checkbox">
                属性:
                    name 定义控件名称, 除了定义名称外, 还起到分组作用
                         一组中的单选按钮或复选框名称必须一致
                    value 值, 尽量提前声明
                    checked 设置预选中,  该属性无值
            4) 隐藏域 & 文件选择框
                隐藏域
                    作用: 想提交给服务器但不想给用户看的数据可以放在隐藏域中
                    语法: <input type="hidden">
                    属性: name定义控件的名称
                          value定义控件的值
                位文件选择框
                    作用: 文件上传时使用
                    语法: <input type="file">
                    属性: name定义控件的名称
            5) 多行文本框
                标记   <textarea></textarea>
                属性:
                    name 定义控件名称
                    cols 指定文本域默认显示的列数
                        一行能够显示的英文字符量, 中文减半
                    rows 指定文本域默认显示的行数
            6) 下拉选择框(下拉列表)
                <select name="变量名">
                    <option value="值1">显示数据</option>
                    <option value="值2">显示数据</option>
                    <option value="值3">显示数据</option>
                </select>
