Python的诞生
    创始人: Guido van Rossum (荷)
    时间: 1989年圣诞节期间
    地点: 阿姆斯特丹

Python的应用领域
    系统运维
    网络编程(搜索引擎, 爬虫, 服务器编程等)
    科学计算
    人工智能, 机器人
    web开发
    云计算
    大数据及数据库编程
    教育
    游戏,图像处理等

Python的优缺点
  优点:
    面向对象OOP(Java, C++, Python, C#, swift, ...)
    免费
    可移植(windows, Linux, unix, ...)
    可混合编程(C/C++/Java...)
    简单易学易用
    开发效率高
    应用领域广泛
    开源
  缺点:
    解释型, 与C/C++相比, 执行速度不够快
    不能封闭源码

Python的官网
    https://www.python.org

Python的版本
    V2.7 (2020年结束维护)
    V3.x
    不会有V4

Python的运行
    Python是解释执行的语言(与C/C++编译的静态语言不同)

Python解释执行器的类型
    CPython (C开发, 默认常用)
    Jython (Java开发)
    IronPython (.net开发)
    Pypy (Python开发)

常用的文件后缀
    .java Java语言后缀
    .c C语言后缀
    .cpp/.cxx/.cc/.C C++语言后缀
    .py Python后缀
    .go Golang后缀

在Python的交互模式下编写并执行代码
进入交互模式:
  $ python3
退出交互模式: 
  $ quit()
  或
  Ctrl + d (输入文件结束符)

Python的注释
    #

Python程序的组成:
    程序是由模块组成的
    模块由语句, 函数, 类等组成
    语句包含表达式
    表达式建立并处理数据对象

                ----- 语句-->表达式-->处理对象
                |
                |
    程序--模块------- 函数
                |
                |
                ----- 类
