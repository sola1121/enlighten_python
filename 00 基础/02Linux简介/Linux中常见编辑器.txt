1.命令行界面文本编辑器 vi/vim
    vi/vim是visual interface的简称, 是Linux中最经典的文本编辑器
    vim是vi的加强版本, 兼容vi的所有指令, 不仅能编辑文本, 而且还具有shell程序编辑的功能.
    1)格式: vi 文件名    (文件存在打开,不存在创建)
    2)vi的三种工作模式
        命令模式 Command mode
            在该模式下, 输入的内容都被编辑器理解为命令, 以命令驱动执行不同的功能.
            进入编辑器, 便处于该模式下.
        输入模式 Insert mode
            也被称为插入模式, 编辑模式, 在该模式下可以编辑文本
        底线命令模式 Last line mode
            以 : 开始, 通常用与文件的保存, 退出
    3)文本写入步骤
        vi 文件名
        命令模式 --> 插入模式
            a 在当前光标的右侧插入
            i 在当前光标的前面去插入
            o 在当前光标的下一行去插入
        编辑完成后, 按ESC键, 由 插入模式 --> 命令模式
        命令模式 --> 命令行模式 shift + :
        在 : (命令行)下输入如下内容如下内容
           :w 保存write
           :wq 保存并退出 write quit
           :q 不保存退出 quit
           :q! 强制退出
        快捷键
            命令模式下
                yy 复制光标所在行 5yy
                dd 删除/剪切光标所在行 8dd
                p (paste)粘贴到光标所在的下一行
                u 回退


2.命令行界面文本编辑器 nano
    1) 使用nano打开文件
        nano 文件名 (文件不存在将会创建)

    2) 基本操作
        移动光标：使用键盘上的方向键。
        编辑文本：就像在其他文本编辑器中一样输入文本。
        删除文本：
        使用 Backspace 键删除光标左侧的字符。
        使用 Delete 键删除光标右侧的字符。
    3) 剪切和粘贴：
        Ctrl+K 剪切光标所在行。
        Ctrl+U 剪切从光标位置到行首的内容。
        Ctrl+6 然后 Shift+Arrow 选择文本，之后按 Ctrl+K 剪切。
        将光标移动到想要粘贴的位置，然后按 Ctrl+U 粘贴。
        查找文本：Ctrl+W，输入要查找的文本，然后按 Enter。
        替换文本：Ctrl+\，输入要查找的文本，然后输入替换文本，选择是否全部替换。
    3) nano的底部栏
        nano 编辑器的底部有几个快捷键提示，它们是：
            ^G：获取帮助（Help）
            ^O：写文件（WriteOut）
            ^R：读文件（Read File）
            ^Y：页面滚动（Prev Page）
            ^K：剪切文本（Cut Text）
            ^C：取消（Abort）
            ^X：退出（Exit）
            ^J：只是写入（WriteOut）
        注:
            这里的 ^ 代表 Ctrl键。
            M代表 Alt键 或者 Option键

    4) 保存和退出
        保存文件：按 Ctrl+O，然后按 Enter 确认文件名。
        退出nano：
        如果文件已保存，按 Ctrl+X。
        如果有未保存的更改，它会询问你是否要保存，按 Y 保存，N 不保存。
        
    5) 配置nano
    nano 的配置文件通常位于 ~/.nanorc。你可以在这个文件中自定义一些设置，比如语法高亮、行号显示等。


3.Sublime编辑器

    Linux中启动命令 $ subl
                  $ subl 文件名
                  $ subl 文件夹

    1.字体放大缩小: Ctrl + 滑轮
                    Ctrl + "+"/"-"
    2.左侧导航栏: view -> side bar -> Show/Hide side bar, Ctrl + k 或 Ctrl + ｂ
    只有打开文件夹时才能打开side bar

    3.同时选中多个光标: Ctrl + 鼠标左键单击
                    
    4.多行注释: Ctrl + /

    5.多选相同的块, 批量修改某个单词: Ctrl + d 重复此操作选下一个单词

    6.新建窗口: Ctrl + n

    7.替换行: Ctrl + Shift + 上下键

    8.删除光标所在行: Ctrl + x

    9.回退: Ctrl + z

    10.控制台:  Ctrl + ` 

    11.换新行 Ctrl + Enter
