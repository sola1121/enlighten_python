1.通配符
  * 匹配任意多个字符(所有)
  ? 匹配一个任意字符
  示例
  ls *.txt
  rm -rf *.txt
  la a?


2.which 查找命令的程序文件所在目录
  Linux命令的本体就是一一个个二进制的可执行程序. 
  这些程序所在的路径都被注册到了执行环境之中, 因此仅需要输入命令名即可执行该程序.
  通过 which 要查找的命令 可以知道指定的命令其本体所在的路径.
  如
    which bluetoothctl
    将打印
    /usr/bin/bluetoothctl
    即该命令所在的目录


3.find 查找文件或目录
  格式: find 起始路径 -size +|-n[kMG]
    + : 表示大于和小于
    n : 表示大小数字
    kMG : 表示大小单位, k表示kb, M表示MB, G表示GB
  如
    查找根目录下小于10kb的文件: find / -size -10k
    查找Home目录下大于1GB的文件: find /home -size +1G 


4.grep 从文件中通过关键字过滤文件行, 并打印
  格式: grep [-n] 关键字 文件路径
    -n : 在结果中显示匹配的行的行号
    -r : 递归搜索文件夹内的文件
    关键字: 必填 表示过滤的关键字, 带有空格或其他特殊符号, 建议用""将关键字包围起来
    文件路径: 必填 表示要过滤内容的文件路径, 可作为内容输入端口
  示例
    过滤little_star文件中含star的行
      grep "star" ~/test/little_star
    过滤 s+(0到多个t)的单词的行
      grep "st*" little_star


5.管道符 |  
  作用: 把管道左边命令的输出, 作为右边命令的输入
  格式: 命令1 | 命令2
  示例
    列出当前目录中的内容, 通过管道将这些内容再由过滤找到"1801"所在行
      ls | grep "1801"
    查看指定文件内容, 通过管道将这些内容再由tail命令输出内容最后8行
      cat /etc/passwd | tail -8
    

6.head 查看文件的前几行
  格式: head -n 文件名   n是要显示的行数, 无此参数默认为10
    head -5 /etc/passwd
    head /etc/passwd


7.tail 查看文件的后几行
  格式: tail -n 文件名   n是要显示的行数, 无此参数默认为10
      -f : 持续查看文件的改变
    tail -5 /etc/passwd
    tail /etc/passwd
  

8.wc 统计文件有多少行
  格式: wc -l 文件名
  示例
    wc -l /etc/passwd


9.date 显示当前系统的时间
  date +%F 显示 年-月-日
  date +%H:%M:%S 显示 时:分:秒   ":"是字符


10.echo 在命令行内输出指定内容
  格式: echo "文本内容"


11.输出重定向 >  把输出到终端的内容输出到指定文件
  格式: 命令 > 文件名
  示例
    echo "Python" > /home/sola1121/python.py
    ls > /home/sola1121/python.py
  注意
    如果文件存在,则覆盖原文件
    如果文件不存在,则创建一个文件
  

12.输出追加重定向 >>  把输出到终端的内容输出到指定文件的末尾
  格式 命令 >> 文件名
  示例
    echo "MySQL" >> Python.py
    cat /etc/passwd >> python.py


13.tar
  1)打包并压缩
    格式: tar -zcvf 新建压缩包名字.tar.gz 要压缩的目录或文件名集合
              -zcvf: zip create view file
          tar -zcvf new.tar.gz little_star
    注意: 压缩包名一般以 .tar.gz 结尾
  2)解压缩
    格式: tar -zxvf 压缩包名字 -C 路径     不用-C参数默认为所在目录


14.gzip, gunzip
  1)gzip   用zip压缩算法对文件进行压缩, 生成压缩后文件
    格式: gzip 文件名
  2)gunzip  对用zip压缩的.gz文件进行解压缩
    gunzip 文件名
  注: 不能压缩目录, 只用于具体文件


20.xxd 可以查看所有文件


21.查看命令或帮助手册
  任何命令都支持 --help选项, 可以通过这个选项, 查看命令的帮助
  如
    ls --help

  如果想要查看命令的更详细的手册, 可以使用man命令  (manual)
  格式: man 命令名
  如
    man cd


数据压缩的种类
  有损压缩
    .mp3 .jpg .mp4 ...
  无损压缩
    .gz .png .rar .xz ...


非正常写法们:
    1)tar xf 压缩包的名字   将会直接解压到当前目录
    2)ll 等同于 ls -al
