1. Linux 命令的基本格式
命令 [选项] [参数]
注意: 命令名区分大小写
	  命令必须有,选项和参数可选

who : 显示登录用户的信息 tty linux的图形界面

pwd : 显示当前的目录路径.
/ : 根代表文件系统起始的位置
路径: 表示一个文件或目录的字符串

ls : 显示指定目录的文件或者子目录信息
常用选项
-l  (long) 以列表形式显示文件的详细信息
-a  (all) 显示全部的文件/目录

路径
绝对路径: 以/(根)开始的路径
相对路径: 不以/(根)开始的路径
	相对路径开始的符号:
	. 当前目录
	.. 上一级目录
	~ 用户主目录

cd : 进入到一个目录(切换到工作目录)
cd 目录名    目录名可以为绝对路径也可以为相对路径
cd .. 切换到当前目录的上一级目录
cd 回家(回到用户主目录) 
cd - 在最近访问的两个目录中来回切换

mkdir (make directory) 创建一个或多个目录
mkdir [选项] 文件夹1 文件夹2 文件夹3...
	常用选项
	-p (parent)逐层创建目录(如果中间的目录不存在,直接创建)

touch : 创建一个或多个文件
touch 文件名1 文件名2 文件名3...

cat : 查看文件内容

rm (remove) 删除一个或多个文件或目录 
	常用选项
	-r 删除文件和目录(包含目录中的内容)
	-i 删除前提示(y表示yes,n表示no)
	-f 强制删除,不给任何提示
生产中常用:
	rm -rf 目录/文件名
	rm -rf * 删除当前目录所有文件
	
2.技巧类

tab键自动补齐,双击提示补齐
	
历史命令,使用↑ ↓键找到已经执行的命令

history : 查看执行过的命令 使用!序号,执行历史命令

3.快捷键操作
虚拟机全屏切换
	ctrl + alt + enter
	
调节终端字体
	放大 Ctrl + shift + '+'
	缩小 Ctrl + "-"
	
清屏
	Ctrl + l
	clear命令
	
退出终端
	exit命令
	
关机
	命令 init 0 或者 shutdown -r now
	
重启
	命令 init 6 或者 reboot
	
终止命令的运行
	Ctrl + C
