1. Linux 命令的基本格式
	command [-options] [parameter]
	命令 [选项] [参数]
	注意: 命令名区分大小写
		命令必须有,选项和参数可选
		在[]中的都是可选的


2. 常见的一些命令

	who : 显示登录用户的信息 tty linux的图形界面
	tty为所在的工作终端

	pwd : (Print Work Directory) 显示当前的目录路径.
	/ : 根代表文件系统起始的位置
	路径: 表示一个文件或目录的字符串

	ls : (List) 显示指定目录的文件或者子目录信息
	直接输入ls, 显示当前目录中的内容
	常用选项
	-l  (long) 以列表形式显示文件的详细信息
	-a  (all) 显示全部的文件/目录

	路径
	Linux的目录结构是树形结构, 但是没有与Windows系统相似的盘符, 而是直接由根目录 / 开始的
	绝对路径: 以/(根)开始的路径
	相对路径: 不以/(根)开始的路径
		相对路径开始的符号:
		. 当前目录
		.. 上一级目录
		~ 用户主目录

		cd : (Change Directory) 进入到一个目录(切换当前的工作目录)
		cd 目录名    目录名可以为绝对路径也可以为相对路径
		cd .. 切换到当前目录的上一级目录
		cd 回家(回到用户主目录) 
		cd - 在最近访问的两个目录中来回切换

	mkdir : (Make Directory) 创建一个或多个目录
	mkdir [选项] 文件夹1 文件夹2 文件夹3...
		常用选项
		-p (parent) 自动创建不存在的父目录, 用于连续创建多层级的目录

	touch : 创建一个或多个文件
	touch 路径名1 路径名2 路径名3...


6.查看文件内容

	cat : 查看文件内容, 将直接把文件内容全部显示出来
		格式: cat 文件名
		示例: cat /etc/passwd
	
	more : 查看文件内容, 支持翻页, 如果文件内容过多, 可以一页一页的显示
		常用操作
			回车 下一行
			空格 下一屏
			q    退出

	cp : (Copy) 复制文件或文件夹
		常用选项
		-r 复制文件夹使用, 表示递归地复制所有内容
		复制文件: cp 源文件 目标文件
		复制目录: cp -r 目录1 目录2   或者    cp -a 源目录 目标目录 (使用-a可以把文件属性也拷贝)

	mv : (Move) 移动文件或文件夹
		mv 被移动的文件或文件夹 将要去的地方
		重命名: mv 原文件名 目标文件名
				mv 原目录名 目标目录名
		剪切: mv 文件名 路径/[新文件名]
			mv 目录名 路径/[新目录名]
  
	rm : (Remove) 删除一个或多个文件或目录 
		常用选项
		-r 删除文件和目录(包含目录中的内容)
		-i 删除前提示(y表示yes,n表示no)
		-f 强制删除,不给任何提示
	生产中常用:
		rm -rf 目录/文件名
		rm -rf * 删除当前目录所有文件
